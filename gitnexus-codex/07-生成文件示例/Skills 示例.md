# Skills 示例

Skill 是任务级工作流。相比 AGENTS.md 的全局规则，Skill 更像“遇到某类任务时按这套步骤做”。

## 常见 Skill

| Skill | 适用场景 |
|---|---|
| gitnexus-exploring | 理解架构、追踪流程 |
| gitnexus-impact-analysis | 修改前做影响面评估 |
| gitnexus-debugging | 追踪错误来源 |
| gitnexus-refactoring | 重命名、拆分、抽取 |
| generated community skills | 某个自动聚类功能区的专用指南 |

## 技术价值
